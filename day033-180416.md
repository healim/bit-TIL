#### HTML 개념

1. Element
2. attribute 

```java
<% for(int i=1; i <= 10; i++){
                out.println("<li>" + i + "번째 행</li>"); // out이 버퍼 식별자
            } %>
```

jsp에서 작성 -&gt; 자바로 : String 취급

out 버퍼에 모음, append됨 - println \(자바에서 버퍼에 추가하고 싶을 때 사용\)

```html
<li>1번째 행</li>
<li>2번째 행</li>
<li>3번째 행</li>
<li>4번째 행</li>
<li>5번째 행</li>
<li>6번째 행</li>
<li>7번째 행</li>
<li>8번째 행</li>
<li>9번째 행</li>
<li>10번째 행</li>
```

버퍼에 모은 내용 브라우저에 전달

```jsp
		<ul>
			<% for(int i=1; i <= 10; i++){
				out.println("<li>" + i + "번째 행</li>");
			} %>
		</ul>
		<ul>
			<% for(int i=1; i <= 10; i++){ %>
				<li> <% out.println(i); %> 번째 행</li>
			<%
				} 
			%>
		</ul>
		<ul>
			<% for(int i=1; i <= 10; i++){ %>
				<li> <%= i %> 번째 행</li>
			<%
				} 
			%>
		</ul>
```
